# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

## [1.0.0] - 2026-03-03

### Added
- **English Localization**: Complete translation of all site content from Portuguese to English for international audience support
- **Modern Design System**: Redesigned header and hero section with contemporary visual elements and smooth animations
- **Blog Module**: Full blog system with Markdown support and dynamic slug-based routing
- **Changelog System**: Integrated changelog page to track and communicate updates to users
- **Documentation**: Comprehensive docs section with multi-level navigation for getting started and essentials guides

### Changed
- **Header & Hero Section**: Updated to premium design with softer color palette and modern typography
- **Animation System**: Improved page transitions and micro-interactions for better user experience
- **Layout Structure**: Refined default layout and responsive design across all pages
- **Content Management**: Consolidated pricing, blog, and changelog content in structured YAML and Markdown formats

### Improved
- **Visual Hierarchy**: Enhanced contrast and spacing throughout the site for better readability
- **Performance**: Optimized component structure for faster rendering and better SEO
- **User Experience**: Smoother scrolling, better navigation flow, and improved accessibility

## [0.1.0] - Initial Release

### Added
- **Project Foundation**: Initial project structure with Nuxt 3, Vue 3, and TypeScript
- **Core Components**: Basic layout components (Header, Footer, Logo) and content sections
- **Landing Page**: Main index page with hero section and key messaging
- **Pricing Page**: Dynamic pricing section with plan comparison
- **Navigation System**: Multi-level routing with auth and docs layouts
- **Configuration**: Setup for Nuxt modules, ESLint, and TypeScript
